Surge5 Configuration

Here is an example Surge5 configuration file you can use:

# Surge 回国配置 (Return-to-China Configuration)
# 适用于海外用户通过中国代理访问国内服务
[General]
loglevel = notify
dns-server = 223.5.5.5, 114.114.114.114
ipv6 = false
[Proxy]
# Example
ChinaServer = http, server-address, port, username, password
[Proxy Group]
China = select, ChinaServer, DIRECT
[Rule]
RULE-SET,https://ruleset.skk.moe/List/ip/domestic.conf,China
FINAL,DIRECT
